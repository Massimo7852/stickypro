Sticky Pro — Support

Need help? Write to massimodelpizzo@gmail.com — we usually reply within 48 hours.

Common questions

  • Where are my notes stored? Locally in ~/Library/Application Support/StickyPro/. If iCloud sync is enabled, also in your private iCloud container.
  • How do I sync between Macs? Make sure both Macs are signed in to the same iCloud account and iCloud Drive is enabled. Notes sync automatically. For one-shot sharing, use the ✈ button (LAN send).
  • How do I assign a keyboard shortcut to "New note from selection"? System Settings → Keyboard → Keyboard Shortcuts → Services → "Sticky Pro: new note" → assign your combo (e.g. ⌘⇧N).
  • How do I use AI actions (⌘J)? Get an API key at console.anthropic.com and paste it in Settings.
  • How do I encrypt a note? Click the palette icon → "Encrypt (Touch ID)". Touch ID will be required to view the note's contents.
  • I lost a note, can it be recovered? If you deleted it from Sticky Pro, it's gone. If you have iCloud sync, check other Macs. Time Machine backup is your last resort.
